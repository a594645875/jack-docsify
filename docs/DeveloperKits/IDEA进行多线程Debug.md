一直没有调试过多线程运动，偶然间发现同时可以debug多条线程。

1. 打断点的时候，在断点上右键，选择**Thread**  
   ![](https://note.youdao.com/yws/public/resource/7d5f5a54817709d3d217939bd99c5806/xmlnote/B812670D022244FD8FC16550D5F05066/910)

2. 然后debug运行，在左下角切换观察的线程  
   ![](https://note.youdao.com/yws/public/resource/7d5f5a54817709d3d217939bd99c5806/xmlnote/AAE15BC6BD6948789B3423E260A7996D/912)

3. 就可以同时debug多条线程的运行过程。
